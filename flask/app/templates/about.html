What is the Auto-Grader?

The Auto-Grader is a web application designed to automatically compile and grade student submissions for programming assignments. It provides immediate feedback on the correctness of submitted code, helping students learn and improve their coding skills more efficiently.

How does it work?

Students submit their code through a simple web interface. The Auto-Grader then compiles and runs the code in a secure, isolated environment, comparing the output against expected results. Scores and feedback are provided directly through the web interface, offering a seamless experience for both students and instructors.
